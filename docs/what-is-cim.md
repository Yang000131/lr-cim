# 半导体 CIM 系统介绍


> CIM(Computer Integrated Manufacturing) 计算集成制造
>
> 提出者：美国学者哈林顿博士
>
> 核心思想：通过信息技术实现制造企业全流程、全要素的集成与优化，将整个制造过程视为一个统一的信息处理系统，从而提升整体效率和竞争力。


## 🏭 核心系统概览

| 系统简称 | 全称           | 核心功能简述                                                        |
| :------- | :------------- | :------------------------------------------------------------------ |
| **MES**  | 制造执行系统   | 生产订单管理、物料跟踪、数据收集与集成平台                          |
| **EAP**  | 设备自动化系统 | 连接 MES 与生产设备，实现自动化控制和数据采集                       |
| **SPC**  | 统计过程控制   | 利用统计方法监控生产过程稳定性与能力，预防质量变异                  |
| **YMS**  | 良率管理系统   | 分析生产数据，定位良率损失根源，推动改进                            |
| **APC**  | 先进过程控制   | 通过模型和实时反馈，对关键工艺参数进行闭环优化控制                  |
| **FDC**  | 故障检测与分类 | 实时监控设备参数，侦测并分类异常，防止缺陷产生                      |
| **RTD**  | 实时派工系统   | 根据工厂实时状态，动态分派生产任务，优化生产效率                    |
| **MCS**  | 物料控制系统   | 控制天车（OHT）、AGV（自动导引车）、RGV（有轨制导车）等自动搬运晶圆 |

**协同工作流：**

1. **MES**从 ERP 接收生产计划，生成工单，并通知**RTD**。
2. **RTD**根据全局状况，决定将某盒晶圆派往设备 A，指令通过**MES**下发。
3. **MCS**接收指令，调度天车将晶圆盒从 Stocker 运送到设备 A。
4. 晶圆盒到达设备 A 后，**EAP**介入，通知设备“有晶圆待加工”，并下发**MES**指定的工艺配方（Recipe）。配方参数可能已由**APC**
   进行了优化调整。
5. 设备开始加工，**EAP**实时采集设备数据并传回**MES**。
6. **FDC**同时监控设备传感器数据，一旦发现异常苗头，立即报警，防止生产出缺陷品。
7. 加工过程中，**SPC**监控关键的工艺输出参数，确保过程稳定。
8. 晶圆加工完成后，会进行测量和检测。所有数据（设备参数、测量结果、检测缺陷等）都被汇集到**YMS**。
9. **YMS**分析这些数据，计算良率，并分析影响良率的主要因素。分析结果反馈给工程师，工程师可能会调整**APC**的模型参数、优化
   **SPC**的监控界限，或安排设备进行维护（可能由**FDC**触发），从而形成一个持续改进的闭环。
