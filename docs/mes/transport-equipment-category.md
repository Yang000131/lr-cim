# Transport Equipment

## TEQP 类别 {#transport-equipment-category}

| 术语        | 基本定义                                         | 核心功能与定位                                                                    |
| :---------- | :----------------------------------------------- | :-------------------------------------------------------------------------------- |
| **Stocker** | 晶圆**自动化存储设备**，通常是一个大型立库。     | 产线的**核心存储枢纽**，负责晶圆载具的自动化存取、管理和调度 。                   |
| **OHB**     | **空中缓存单元**，一种安装在空中的临时存储装置。 | 产线的**临时“快取区”**，为 OHT 天车提供临时缓冲，减少机台等待时间 。              |
| **Shelf**   | Stocker 或货架上的**一个具体储位**。             | Stocker 内的**最小存储单元**，每个 Shelf 有唯一地址，用于定位一个晶圆盒（FOUP）。 |

## 💡 深入解读

- **Stocker：智能仓库**
  。它不仅仅是存放东西的地方，更是一个高度自动化的智能仓库。它通过精密的机械臂（定位精度可达 ±0.05mm）、RFID 或视觉识别系统，在计算机系统（MCS）的指挥下，实现晶圆盒的自动存入和取出。它能维持 Class
  100 甚至 Class 10 的高洁净度环境，并通过氮气吹扫（N2 Purge）等技术保护对氧气敏感的晶圆，是保障晶圆良率和生产效率的核心

- **OHB：空中接力站**。你可以把它想象成高速公路上的临时应急车道或快递中转点。当物料需要在天车（OHT）和加工机台之间快速中转，或者在天车轨道沿线需要临时存放点时，OHB 就发挥了作用。它的主要目的是
  **消除等待，让物料流转更顺畅**，从而提升整个产线的设备利用率（稼动率）

- **Shelf：最小存储单元**。Shelf 是物理存储的最小单位。在软件系统（如 MCS）中，每个 Shelf 都有其唯一的坐标地址。当系统下达指令时，Stocker
  的机械臂就是通过定位到具体的 Shelf 位置来完成存取的

## 🔧 协同工作模式

在实际的半导体产线中，这三者紧密协作，形成一个高效的物流网络：

1. **任务下达**：制造执行系统（MES）产生一个生产指令，例如“将 Lot A 的晶圆送到光刻机 X 进行加工”。
2. **路径规划**：物料控制系统（MCS）接收到指令，定位到 Lot A 所在的晶圆盒（FOUP）正存放在某个 Stocker 的特定**Shelf**（储位）上。
3. **物料提取**：MCS 命令 Stocker 的机械臂前往对应的 Shelf 取走 FOUP，并将其放置在与 OHT 天车的对接端口（OHT Port）。
4. **空中运输**：OHT 天车取走 FOUP，将其运往目标机台。
5. **缓冲协调**：如果目标机台暂时忙碌，OHT 天车可能会先将 FOUP 临时存放在机台旁或轨道沿线的**OHB**（空中缓存单元）中，待机台就绪后再进行投放
